This section presents briefly some academic projects.

Little robot.

In order to apply what we had learned during the practical sessions in Electronics for embedded systems we were challenged to program and control a two-wheeled robot. For this project to be considered a success, two conditions had to be met. Firstly, the robot had to be able to follow a path on its own, making sure to detect obstacles, and secondly, it had to be controllable by remote control. In groups of three, we worked for two months to ensure that our robot passed both the manual and autonomous demonstrations.

Little adventure game.

I develop a little adventure game using Java in order to apply the core concept of that programming language.
